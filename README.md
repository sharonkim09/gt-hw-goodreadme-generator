# Good Readme Generator 

## Description
This is a command-line application that dynamically generates a good README.md from user's input. 


## Table of Contents
* [Installation](#installation)
* [Usage](#usage)
* [Credits](#credits)
* [License](#license)
* [Contributing](#contributing)

## Installation
1. Clone the repo
2. Open index.js in terminal 
3. Run  node index.js
